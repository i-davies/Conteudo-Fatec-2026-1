# Conteudo

## Pré-requisitos

- [uv](https://github.com/astral-sh/uv) instalado.

## Como Rodar

1. **Instalar dependências**:
   ```bash
   uv sync
   ```

2. **Executar o projeto**:
   ```bash
   uv run main.py
   ```

3. **Rodar a documentação (Opcional)**:
   ```bash
   uv run mkdocs serve
   ```
